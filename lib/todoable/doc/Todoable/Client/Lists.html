#get_list(id:) ⇒ Hash

Fetches a List from the Todoable server, including its Items.

Takes either an arguments Hash, including the id of the List, or a List object itself.

By default this API does not include the List's id in the Hash. We add it here for consistency.

Examples:

Todoable::Client.get_list(id: "41cf70a2-...") #=>

def get_list(id:)
  list = get(path: "lists/#{id}")
  list["id"] ||= id

  list
end

#update_list(id:, name:) ⇒ Hash

Saves changes to a given List to the Todoable server.

Takes either an arguments Hash, including the id of the List and its new name, or a List object itself.

If a List object is passed, the name will be updated to the current name of the List object.

Examples:

Todoable::Client.update_list(id: "123-abc", name: "Birthday List") #=>
  {"name"=>"Birthday List", "src"=>"...", "id"=>"123-abc"}

Parameters:

  • id (Symbol) —

    the id of the List

  • name (Symbol) —

    the new name of the List

def update_list(id:, name:)
  path = "lists/#{id}"
  params = {
    "list" => {
      "name" => name
    }
  }
  request(method: :patch, path: path, params: params)
end
