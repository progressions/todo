#finish! ⇒ Boolean

Mark this Item as finished on the Todoable server, and re-initialize this Item object by reloading its associated List and finding its new attributes.

We reload the List and its Items in order to correctly set the finished_at attribute on this Item object so it matches the server.

Raises an exception if the Item is already finished.

  true

def finish!
  raise ItemAlreadyFinished.new("Item: `#{name}` is already finished") if finished?

  if self.class.finish(list_id: list_id, id: id)
    attributes = list.reload.attributes["items"].find { |i| i["id"] == self["id"] }
    initialize(attributes)

    true
  else
    false
  end
end
